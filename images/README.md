# Jeek Lights - Website Assets Guide

This directory contains the image assets for the Jeek Lights single-page premium website. Place the corresponding high-resolution assets from the brochure in these paths.

> [!NOTE]
> The website's source code contains built-in, elegant CSS/SVG fallbacks. Even if these directories are empty, the website will load and function with clean styled placeholders.
